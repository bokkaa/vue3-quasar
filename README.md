# Vue3 Firebase App

## 기간 
- 2024.02.03 ~ 2024.03.10

## 기능
- 간단한 커뮤니티 사이트
- 구글 로그인, 이메일 로그인, 프로필 수정, 비밀번호 수정 등의 기능을 제공한다.
- 게시판 글 작성(사진 및 해시태그 등록기능) 및 검색 기능을 제공한다.
- 회원탈퇴 기능이 추가되었다(2024.03.11)
